# Computer Hero

Website en tools voor **Computer Hero** (chero.nl) — computerhulp aan huis in de regio Utrecht.

## Over Computer Hero
Computerhulp aan huis voor particulieren in de regio Utrecht, Amersfoort, Hilversum, Zeist, Maarssen, Nieuwegein, Soest, Vianen en Bilthoven.

## Tech Stack
- Static HTML + Bootstrap 5
- Pug templates (src/templates/)
- SCSS (src/scss/)
- HubSpot chat + meetings integratie
- Formspree contactformulier
- MailChimp nieuwsbrief
